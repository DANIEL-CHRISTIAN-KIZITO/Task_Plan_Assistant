#App Description

TaskPlanAssistant is a simple Flutter-powered mobile application designed to help users manage daily tasks with additional notes.
The app allows users to:

Create tasks

Add detailed notes

Mark tasks as completed

Edit or delete tasks

Organize tasks for better productivity

#The goal of this app is to provide a clean, minimal, and efficient task-management experience for students and general users.

#How to Run the Project

Follow these steps to run the app locally:

1. Install Flutter

Make sure Flutter SDK is installed and added to PATH.
Check using:

flutter doctor

2. Navigate to the project folder
cd Task_Plan_Assistant

3. Install dependencies

Run:

flutter pub get

4. Connect a device or start an emulator

Check available devices:

flutter devices

5. Run the application
flutter run

Optional: Run with hot reload

In the terminal running your app, press:

r   → hot reload
R   → hot restart
